Este projeto foi desenvolvido como parte do **Desafio 01 da Formação Lógica de Programação** da [DIO](https://www.dio.me/curso-logica-programacao).

## 🚀 Objetivo
O objetivo é aplicar conceitos fundamentais de programação, como:
- Variáveis e Tipos de Dados
- Operadores de Comparação
- Estruturas de Decisão (`if`, `elif`, `else`)
- Entrada e Saída de Dados

## 🎮 Contexto (MLBB)
O programa recebe o nome de um Player e a sua quantidade de estrelas, classificando-o nos elos oficiais do jogo Mobile Legends, desde **Warrior** até o lendário **Mythical Immortal**.

## 🛠️ Como rodar o projeto
1. Certifique-se de ter o Python 3 instalado.
2. Clone este repositório.
3. No terminal, execute:
   ```bash
   python3 desafio.py
